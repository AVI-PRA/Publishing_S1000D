CHAPTER – 1

INSPECTION AND MAINTENANCE ROUTINE

1.1 INTRODUCTION

1.1.1 The design of the BRAHMOS Fire Control System Mark-II (BFCS Mk-II) is modular in nature to reduce the MTTR. Various operations have been realized by adopting combined software - hardware approach. Standard reliable computer board and other PCBs / modules have been used to enhance the MTBF. There are hardly any components / parts that need regular maintenance. However, it is recommended that the user should carry out maintenance routines described in the succeeding paragraphs to keep the system operational.

1.1.2 Following precautions / protective measures are to be followed while handling, operating, and performing maintenance operation on the BFCS Mk-II:

(a) Ensure power to the unit is switched OFF while inserting or removing cards / modules.

(b) Ensure that all the cards / modules are inserted in the correct slots.

(c) Ensure that components mounted on cards / modules should not be touched with bare hand. Always wear anti-static wrist band while performing operations on these cards / modules.

(d) Ensure that all the cards are packed in anti-static covers.

(e) Ensure that all spares packed properly and kept in a dry and controlled environment.

(f) Use appropriate tools while handling / maintenance.

(g) Use appropriate rating fuse.

(h) Use only spares supplied in On Board Spares (OBS).

(j) Do not connect or remove cables in power ON conditions.

(k) All connectors should be covered with corresponding dust caps to prevent damage to the pins, when left de-mated.

(l) Always maintain a break of 15 minutes if the system is to be powered ON again after it has been switched OFF. However, this action should not be done frequently.

(m) Keeping the system powered ON continuously for a period of more than 168 hours is not recommended.

(n) Exposing the cards in extreme humid conditions for long time is to be avoided.

(p) While transporting the cards / modules, the same are to be wrapped in bubble sheet over the anti-static cover and vertically placed in a hard container. cards are not to be stacked.
